## Sobre o Slim Up!

Slim Up! é um aplicativo voltado ao bem-estar, ajudando o usuário a:

- Organizar a ingestão diária de líquidos;
- Programar pausas para relaxamento;
- Definir lembretes para exercícios físicos.

As páginas deste repositório têm caráter exclusivamente informativo e jurídico, para cumprimento de requisitos de publicação em lojas de aplicativos.
